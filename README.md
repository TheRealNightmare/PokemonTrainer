### Installation & Setup:

1. Clone or Download the Project & navigate to the project directory:
```sh
cd poketeam_project
```

2. Set Up Virtual Environment

```sh
python -m venv .venv
source .venv/bin/activate
```

3. Install Dependencies

```sh
pip install -r requirements.txt
```
5. make the .env file
```sh
cp .env.example .env
```

5. Run the app
```sh
python app.py
```

Open your browser and visit: http://127.0.0.1:5000
